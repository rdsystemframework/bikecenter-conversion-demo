# Bikecenter Conversion Demo

Dieses Projekt zeigt eine conversion-optimierte Website-Struktur für einen Rennrad-Fachhändler.

Die Seite ist als Demo aufgebaut, um zu zeigen, wie eine moderne Händlerseite für Rennräder aussehen kann – mit Fokus auf:

• Probefahrt-Anfragen  
• Rennrad-Präsentation  
• Store-Impressionen  
• Service & Beratung  
• klare Call-to-Actions

---

## Ziel der Demo

Viele Händler-Websites sind technisch gewachsen und verlieren dadurch an Übersicht und Conversion.

Diese Demo zeigt eine alternative Struktur mit Fokus auf:

1. Klarer Einstieg (Hero-Bereich)
2. Schnelle Produktorientierung
3. Emotionale Rennrad-Bilder
4. Beratung und Service
5. Termin / Probefahrt als Hauptziel
